📚 COMMON QUERIES
──────────────────────────────────────────────────────────────────

List all tables:
  SELECT * FROM information_schema.tables WHERE table_schema='public';

Get table sizes:
  SELECT tablename, pg_size_pretty(pg_total_relation_size(schemaname||'.'||tablename))
  FROM pg_tables WHERE schemaname='public' ORDER BY pg_total_relation_size DESC;

View Audit Logs (latest first):
  SELECT user_id, action, data_type, timestamp FROM audit_logs ORDER BY timestamp DESC;

Get Overdue Subject Rights Requests:
  SELECT id, subject_id, request_type FROM subject_rights_requests 
  WHERE status='PENDING' AND deadline < NOW();

Daily Audit Summary:
  SELECT * FROM audit_log_summary_daily;
