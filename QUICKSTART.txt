║                          USEFUL COMMANDS                                   ║
╚═══════════════════════════════════════════════════════════════════════════╝

Windows Command Prompt:
  gobernador.bat              Start all services
  gobernador.bat stop         Stop all services
  gobernador.bat restart      Restart all services
  gobernador.bat logs         View live logs (Ctrl+C to exit)
  gobernador.bat status       Show container status
  gobernador.bat cleanup      Delete all data (⚠️ WARNING)

macOS/Linux Terminal:
  ./gobernador.sh             Start all services
  ./gobernador.sh stop        Stop all services
  ./gobernador.sh restart     Restart all services
  ./gobernador.sh logs        View live logs (Ctrl+C to exit)
  ./gobernador.sh status      Show container status
  ./gobernador.sh cleanup     Delete all data (⚠️ WARNING)

╔═══════════════════════════════════════════════════════════════════════════╗
║                        TROUBLESHOOTING                                     ║
╚═══════════════════════════════════════════════════════════════════════════╝

❌ Port Already in Use?
   Windows: netstat -ano | findstr :5173
   Mac/Linux: lsof -ti:5173 | xargs kill -9

❌ Services Won't Start?
   Check logs: gobernador logs
   Usually takes 30 seconds for first start

❌ Database Connection Failed?
   Stop: gobernador stop
   Wait 5 seconds
   Start: gobernador start
   Wait 30 seconds

❌ Clear Everything?
   gobernador cleanup
   Then: gobernador start
